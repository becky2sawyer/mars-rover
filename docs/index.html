    Welcome to marsrover

    화성탐사선 marsrover 입니다. - 탐사의 목적은 감자 키우기 - 지구와 핑퐁 통신하기

    marsrover

    marsrover 모듈.

    화성에서 감자를 키우기 적합한 곳을 찾아 지구와 핑퐁 통신을 합니다.

    Examples:

    marsrover 사용법은 아래와 같습니다.

    $ pip install ./
$ marsrover-ping
$ marsrover-ping kr
$ marsrover-ping en

    en 또는 kr 로 통신 언어 설정이 가능합니다.

    Attributes:

    Name Type Description
    MSG list

    통신가능 언어팩 이 담겨 있다. 한국어 및 영어 통신이 가능하며, 가각 kr, en 으로 설정 가능하다.
